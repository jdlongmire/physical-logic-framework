## Common Issues & Solutions

### Notebook Issues
| Issue | Solution | Prevention |
|-------|----------|------------|
| Memory error N>8 | Limit to N≤6 for enumeration | Use sampling methods |
| Missing ./outputs/ | `os.makedirs('./outputs', exist_ok=True)` | Check notebook setup |
| Slow linear extensions | Add `limit=1000` parameter | Optimize algorithms |
| NetworkX conflicts | Use `nx.all_topological_sorts()` v2.6+ | Update dependencies |

### Lean Issues  
| Issue | Solution | Prevention |
|-------|----------|------------|
| Build failures | Check Lean version: v4.23.0-rc2 | Use lean-toolchain file |
| Missing mathlib | `lake exe cache get` | Proper lakefile.toml |
| Type errors | Check imports and dependencies | Follow build order |
| Sorry warnings | Expected for computational proofs | Complete proofs iteratively |

### Integration Issues
| Issue | Solution | Prevention |
|-------|----------|------------|
| Result mismatches | Cross-validate numerical values | Systematic verification |
| Conceptual gaps | Study both frameworks together | Use cross-reference protocol |
| Performance bottlenecks | Run frameworks independently first | Optimize workflow |
